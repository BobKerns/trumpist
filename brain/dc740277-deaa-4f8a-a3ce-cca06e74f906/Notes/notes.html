By Thread

By Thread is intended to allow you to trace through complex series of related events.

It is still in the early stages, as I focus on filling in the needed events.

Threads all use the same heavy orangish link type, and are labelled with one or more thread labels.

Thread labels allow you to see what thread the connection is part of.

A Thread link implies a Time link since you cannot have two links between the same two nodes.

For the same reason, a Thread link may be labelled with more than one thread label.
